# Bulk Print Guide

Use Bulk Print when you need to print tickets or receipts for many paid bookings at once.

## Open Bulk Print

1. Log in to the admin panel.
2. Open **Bulk Print** from the left menu.
3. Choose the department types you want:
   - Regular Bingo
   - Special Bingo
   - Live Event / Venue
4. Choose the **From Date** and **To Date**.
5. Click **Load Tickets**.

## Choose What To Print

After tickets load, all tickets are selected by default.

Use the checkboxes beside each ticket to remove anything you do not want to print.

Use the filter if needed:

- **All tickets**: shows everything for the selected dates.
- **Unprinted only**: shows tickets that have not been marked printed.
- **Printed only**: shows tickets already marked printed.

## Print Special Paper

Click **Print Special Paper** when printing the main paper tickets.

Use this for:

- Regular bingo tickets
- Special bingo tickets
- Live event / venue tickets

Regular and special bingo print 3 tickets per page. Live event / venue tickets print 6 tickets per page.

If the last page has blank spaces, that is normal. The system keeps the layout aligned for physical paper.

## Print Thermal Copy

Click **Print Thermal Copy** when printing receipt-style copies on the thermal receipt printer.

Thermal copies are grouped by booking, using the selected tickets.

Before printing, choose the receipt settings:

- Paper width: 58mm or 80mm
- References
- Table / Seat
- Prices
- Add-ons
- Timestamp
- Cut after each receipt: no cut, 50%, 70%, or 90%

Click **Save Settings** if you want to keep those receipt settings for next time.

For the Epson TM-T88V, choose a cut percentage when you want every bulk thermal receipt to finish as its own cut-ready page. Match the same cut percentage in the printer driver so it leaves the small attached edge instead of cutting the paper completely.

## Mark Tickets Printed

After printing, click **Mark Selected Printed** if you want the system to remember those tickets were printed.

This helps later because you can use **Unprinted only** or **Printed only** to avoid reprinting the wrong tickets.

## Common Workflow

1. Select the date range.
2. Click **Load Tickets**.
3. Uncheck any tickets that were already printed.
4. Click **Print Special Paper** for the main tickets.
5. Click **Print Thermal Copy** if receipt copies are also needed.
6. Click **Mark Selected Printed** after printing is done.

## Quick Notes

- Bulk Print only loads paid bookings.
- Archived or past sessions are included.
- If no tickets show up, check the selected date range first.
- Blank spaces in print preview usually mean the page is holding room for the fixed paper layout.
